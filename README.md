# Toss Payments Admin Plugin (fe-admin)

Two admin views:
- `/admin/toss/payments` — payments list + refund.
- `/admin/toss/cash-receipts` — cash receipt ledger (shows only
  identifier hashes, never plaintext phones/biz-IDs).

Pairs with [`vbwd-plugin-toss-payments`](https://github.com/VBWD-platform/vbwd-plugin-toss-payments).

**Core:** [vbwd-fe-admin](https://github.com/VBWD-platform/vbwd-fe-admin)
